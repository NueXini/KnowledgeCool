# KnowledgeCool

知识库，这是一个简单的文库下载器后端 (C++练手项目)

# Support

|         文库         |                         官网                         |
| :------------------: | :--------------------------------------------------: |
| 银符在线考试模拟题库 |   [https://www.yfzxmn.cn/](https://www.yfzxmn.cn/)   |
|    百度文库高校版    | [https://eduai.baidu.com/](https://eduai.baidu.com/) |
|         知网         |    [https://www.cnki.net/](https://www.cnki.net/)    |

# API

## 银符在线考试模拟题库

`POST /api/yfzxmn`

**请求参数：**

| 参数 | 类型 | 描述   |
| ---- | ---- | ------ |
| id   | int  | 试卷id |
